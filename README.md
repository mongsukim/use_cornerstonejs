# 프로젝트 설명
cornerstone.js 라이브러리를 활용하여 의료 영상 데이터(DICOM)을 볼 수 있다.  

마우스를 사용해   
```
(1) windowing 조절  
(2) pan/zoom  
(3) 길이 체크
```

기능을 사용할 수 있다.
